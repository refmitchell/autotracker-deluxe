Open the autotracker directory in the terminal (as you did during the setup).

Startup

To start the software, run:

$ ./dtrack2

This will open the main window

The window is split into two main panels. The project panel (upper) and the tool panel (lower).

Project panel

The current project directory is shown in the central text entry. This should persist across software restarts (i.e. DTrack2 will remember which project you were last working on). If this is the first time you've started the software then it will be blank.

Creating a new project

Clicking 'New project' will display a file dialog asking you to specify where you want this to be stored. Navigate to the folder where you want to create your project, type the name of the new project in the filename textbox, and click ok.

Selecting an existing project

Clicking 'Select project' will allow you to select an existing project. The selector is a little clunky, you navigate to the desired project directory then click ok.

Tool panel

In order to run a tool, you select the corresponding radio button, then click 'Run'. Each tool has its own page:

  1. Choose video files
  2. Configure calibration board
  3. Calibration manager
  4. Run autotracker
  5. Process tracks
