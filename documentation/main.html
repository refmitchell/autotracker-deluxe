DungTrack 2 documentation

DungTrack 2 provides autotracking and basic track analysis for dung beetle orientation experiments. This software is designed for arena-based experiments. It will work for other types of experiment but the feature set, user interface, and available analysis are designed for arena-based experiments.

This documentation is written in basic HTML so that

  1. Instructions can be divided into different linked pages.
  2. Everyone using the software will have access to software which can render the documentation (a web browser).
  3. Updating the source is relatively straightfoward (as easy as Markdown).
It's not pretty but it's functional.
