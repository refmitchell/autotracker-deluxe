From the main window, clicking Options in the lower right-hand corner will display this window

Your options may be configured differently!

This window allows you to change the way the different tools operate. Options for a given tool are documented on that tool's page:

  1. Video selection tool
  2. Calibration board configuration tool
  3. Autotracker
  4. Plotting and analysis tool

The calibration manager does not currently have any configuration options.

These options are saved to params.json in your software directory. They will persist across different tracking sessions but may be undone if you download a new version of the software by downloading the zip file from GitHub (using Git should preserve your settings).
