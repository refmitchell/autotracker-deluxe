Installing dependencies

If you're reading this, I presume you've already downloaded the software repo.
  1. Make sure you've installed python and pip (available here).
  2. To open a terminal (command prompt) in the software directory, open your file explorer (Win+E) and navigate to the repository you downloaded from GitHub.
  3. Click on the address bar (or press Alt+D) and type "cmd" then hit Enter. A terminal window should open.
  4. Run: $ pip install -r requirements.txt

Running the software

With the terminal open in the software folder, run: $ py src\main.py
