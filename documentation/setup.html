Opening a terminal in the correct place

If you already know how to use the terminal, switch into the autotracker-deluxe directory and jump to "Installing dependencies" below.

MacOS

Open Finder and navigate to the folder you downloaded from github. Hold control and click on the folder, then select 'Services' → 'New Terminal at Folder'. A terminal window should open.

Linux (Ubuntu)

Open your file explorer, find the folder you downloaded, right click and select 'Open in Terminal'. This may be distribution dependent.

Installing dependencies

The dependencies can be installed using a bash script. Make sure the terminal is open in the autotracker directory, then run:

$ ./SETUP

This script will:

  1. Check for Python 3 and Pip 3 installations
  2. Create a virtual environment (a sandbox in which you can install the dependencies without affecting other software which is installed)
  3. Install the dependencies within this virtual environment
  4. Create a launcher for the tracking software
If you open the SETUP file in a text editor you can see what this script does.

Note for Linux users

If working with linux then you need to make sure python3-venv and python3-tk are installed. On Ubuntu you can achieve this by running:

$ sudo apt install python3-venv python3-tk

Setup and launcher scripts

To ease usage, python, pip, and virtual environment management are wrapped within bash scripts. If you don't want to run these scripts, then you can install the dependencies as normal using:

$ pip3 install -r requirements.txt

and you can run the software with:

$ python3 main.py

If you're trying to use the setup script and it doesn't work then please post an issue!
